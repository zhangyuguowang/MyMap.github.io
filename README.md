# 2023年中国经济与人口数据可视化平台

这是一个基于Web的数据可视化项目，展示2023年中国各省份的GDP、人均GDP和人口密度数据。

## 项目特点

- **多维度可视化**：包含专题地图、复合饼图和条形图三种可视化类型
- **交互式体验**：支持点击交互、图例切换、视图控制等功能
- **响应式设计**：适配桌面端和移动端设备
- **数据完整性**：涵盖中国34个省级行政区的最新数据

## 包含的可视化内容

1. **综合专题地图**：展示人口密度（面状底色）和人均GDP（紫色圆点）的空间分布
2. **区域人均GDP饼图**：六大经济区域及下属省份的人均GDP分布，支持层级钻取
3. **人口密度Top10条形图**：人口密度最高的10个省份/地区对比

## 技术栈

- **前端框架**：原生HTML5/CSS3/JavaScript
- **可视化库**：ECharts 5.4.3
- **图标库**：Font Awesome 6.4.0
- **部署平台**：GitHub Pages

## 使用说明

1. 直接打开`index.html`即可在浏览器中查看
2. 使用顶部的导航栏在不同可视化图表之间切换
3. 在专题地图中，可以使用控制按钮切换图例类型、隐藏/显示省份名称和人均GDP圆点
4. 在区域人均GDP饼图中，点击区域可查看该区域下各省份的详细数据
5. 点击"全屏"按钮可获得更好的浏览体验
