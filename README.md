# 2023年中国经济与人口数据可视化平台

这是一个基于Web的数据可视化项目，展示2023年中国各省份的GDP、人均GDP和人口密度数据。项目采用现代化设计，提供丰富的交互体验和响应式布局。

## 项目特点

- **多维度可视化**：包含专题地图、复合饼图和条形图三种可视化类型
- **交互式体验**：支持点击交互、图例切换、视图控制、自定义筛选等功能
- **创新性功能**：
    专题地图：可切换离散/连续图例，隐藏/显示标签和圆点
    复合饼图：支持区域-省份层级钻取，点击查看详细数据
    交互式条形图：支持自定义选择省份，三种显示模式自由切换
- **响应式设计**：适配桌面端、平板和移动设备
- **数据完整性**：涵盖中国34个省级行政区的2023年度数据

## 包含的可视化内容

1. **综合专题地图**：
    面状底色表示人口密度（颜色越红表示密度越高）
    紫色圆点表示人均GDP（圆点越大表示人均GDP越高）
    支持切换离散/连续图例，可拖动图例调整显示范围
2. **区域人均GDP饼图**：
    六大经济区域人均GDP分布概览
    点击区域可查看该区域下各省份的详细数据
    支持层级钻取，点击任意位置返回区域视图
3. **交互式人口密度条形图**：
    支持自定义选择，自由选择想要对比的省份
    提供排名快捷键，如Top5排名模式；Top10排名模式
    特殊数据处理，港澳特殊行政区人均GDP与人口密度数值特殊处理
    智能颜色方案，自动调整显示方式

## 技术栈

- **前端框架**：原生HTML5/CSS3/JavaScript
- **可视化库**：ECharts 5.4.3
- **图标库**：Font Awesome 6.4.0
- **依赖管理**：CDN引入，无需本地安装
- **部署平台**：GitHub Pages
